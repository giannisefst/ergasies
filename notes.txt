Θέμα 2
Ο στόχος της εργασίας είναι να γραφεί μία διαφορετική “τρίλιζα”, όπου οι 2 παίκτες παίζονται από χρήστες. 
Ο κάθε παίκτης έχει 6 κομμάτια διαφορετικού μεγέθους (2 μικρά, 2 μεσαία και 2 μεγάλα) και μπορεί να καλύψει κομμάτια δικά του ή του αντιπάλου για να κάνει τρίλιζα.
Το παιχνίδι τελειώνει όταν δεν μπορεί να παιχτεί άλλο κομμάτι. Δείτε το video (https://www.youtube.com/watch?v=HxF_vnr34i8) για το πως παίζεται το πλήρες παιχνίδι.

Θέμα 11

Σύγκριση δύο προφίλ στο Twitter. Γράψτε ένα πρόγραμμα το οποίο παίρνει από το χρήστη το όνομα δύο χρηστών του Twitter.
Στη συνέχεια, επιλέγει από το καθένα τους αριθμούς TWEETS, FOLLOWING, FOLLOWERS και LIKES. 
Για κάθε μία κατηγορία γίνεται μία σύγκριση όπου το κάθε προφίλ παίρνει ένα σκορ 0 αν είναι ίδιο ή μικρότερο
με την αντίστοιχη τιμή του άλλου προφίλ και 1 αν είναι μεγαλύτερη. Στο τέλος εμφανίζει το σκορ.

Παράδειγμα για δυο προφίλ:

    Προφίλ @SouthPark TWEETS 11.4K, FOLLOWING 281, FOLLOWERS 1.83M, LIKES 3985
    Προφίλ @SouthPark TWEETS 20.5K, FOLLOWING 7776, FOLLOWERS 147K, LIKES 21.8Κ

Σκορ 1-3

Notes Exercise 11:

In order to run the Twitter application you need install 'tweepy' library. Please follow the below steps:
1. Go to python istallation folder. Find folder 'scripts' and copy the entire path. 
2. Open a windows command line and 'cd' the above folder. Then type "pip.exe install tweepy".
